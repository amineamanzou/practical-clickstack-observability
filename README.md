# Practical ClickStack Observability Journey

This repository acts as a comprehensive knowledge reference designed to support training on the ClickHouse website. It serves as a searchable and navigable companion, allowing you to quickly look up concepts, extensive notes, and technical details while you learn.

## Purpose

Use this repository to:

- **Search** for specific ClickHouse topics and configurations instantly.
- **Navigate** through structured documentation alongside your official training.
- **Reference** verified notes and architectural insights.

## Methodology

This project is built as a **digital garden** utilizing:

- **Hugo**: For a fast, static-site experience.
- **Hextra Theme**: Providing a modern documentation layout.
- **Zettelkasten Methodology**: Notes are atomically written and interconnected, fostering a network of understanding rather than just linear information.

## Usage

The site is designed to be deployed and viewed via GitHub Pages, serving as an always-available reference tool during your studies.

## local development

To run the site locally:

```bash
hugo server
```

Then access it at `http://localhost:1313`.
